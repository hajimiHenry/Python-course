# Python-100-Days 前15天题单（26题）

说明：本题单覆盖 `Day01 ~ Day15` 的核心内容，按“基础 -> 进阶 -> 综合”排列。建议每题独立成一个 `.py` 文件完成，先不看答案，做完再自测。

## A. 基础语法与流程控制（1-9）

1. `Hello, Python`  
   编写程序输出三行内容：姓名、学习目标、今天学习时长（小时）。

2. 变量与类型转换  
   输入商品单价和数量，计算总价并输出，要求总价保留两位小数。

3. 运算符综合  
   输入两个整数 `a` 和 `b`，分别输出它们的加、减、乘、整除、取模、幂运算结果。

4. BMI 判断  
   输入身高（米）和体重（公斤），计算 BMI，并按区间输出“偏瘦/正常/超重/肥胖”。

5. 成绩等级  
   输入百分制成绩，使用 `if-elif-else` 输出等级：A/B/C/D/E。

6. 闰年判断  
   输入年份，判断是否闰年（规则：能被4整除但不能被100整除，或能被400整除）。

7. 1~100 累加题  
   分别用 `for` 和 `while` 计算 1 到 100 的和。

8. 九九乘法表  
   使用嵌套循环打印标准九九乘法表。

9. 素数统计  
   输出 2~100 的所有素数，并统计总个数。

## B. 分支循环实战 + 数据结构（10-18）

10. 斐波那契前 n 项  
    输入 `n`，输出斐波那契数列前 `n` 项（`1, 1, 2, 3...`）。

11. 水仙花数  
    找出 100~999 范围内所有水仙花数并输出。

12. 列表基础操作  
    给定列表 `scores = [88, 76, 90, 100, 67]`，完成：添加一个分数、删除最低分、输出平均分。

13. 列表推导式  
    生成 1~50 中所有偶数的平方列表，并输出前 10 个元素。

14. 元组拆包  
    有元组 `("Alice", 20, "Python")`，通过拆包分别拿到姓名、年龄、方向并格式化输出。

15. 字符串处理  
    输入一个字符串，输出：长度、反转字符串、是否回文（忽略大小写）。

16. 字符统计  
    输入一段字符串，统计并输出其中大写字母数、小写字母数、数字数、空格数。

17. 集合运算  
    两个班级选修 Python 的学生名单（用集合表示），输出交集、并集、差集。

18. 字典综合  
    用字典保存 5 名学生成绩，完成：查询某人分数、修改分数、输出最高分学生。

## C. 函数与应用实战（19-26）

19. 最大值函数  
    定义函数 `max_of_two(a, b)` 返回较大值，并写 3 组测试数据验证。

20. 阶乘与组合数  
    定义 `factorial(n)` 和 `comb(m, n)`，计算组合数 `C(m, n)`。

21. 素数函数  
    定义 `is_prime(num)`，再用它输出区间 `[m, n]` 内所有素数。

22. 最大公约数与最小公倍数  
    定义 `gcd(x, y)` 与 `lcm(x, y)`，并验证两组输入。

23. 随机验证码  
    定义函数 `generate_code(*, code_len=4)`，生成数字+字母验证码，打印 5 组结果。

24. 描述性统计函数  
    依次实现 `mean / median / var / std`（可选再做 `cv`），对给定列表输出统计值。

25. 双色球选号函数化  
    封装 `choose()` 与 `display()`，实现随机生成 `n` 注号码并格式化显示。

26. 综合小项目：学生成绩管理  
    使用“循环 + 分支 + 列表/字典 + 函数”实现命令行菜单程序，支持：
    - 添加学生成绩
    - 查询学生成绩
    - 修改学生成绩
    - 删除学生
    - 按成绩从高到低显示全部学生

## 建议的完成顺序

`1-9`（夯实语法） -> `10-18`（数据结构） -> `19-26`（函数化与综合）

## 0. 先记住 3 句话（够你做题了）

1. 字符串就是“文本”，比如 `"hello"`、`"你好"`。
2. 字符串**不可变**：你不能在原地改它，只能得到一个“新字符串”。
3. 大部分字符串方法都“返回新值”，要么 `print(...)` 看结果，要么用变量接住。

---

## 1. 字符串是干什么的

一句话：**处理文字信息**。

比如：
- 查找关键词（有没有 `"py"`）
- 修改文本（把 `"world"` 换成 `"python"`）
- 拆分文本（按 `#` 拆开）
- 拼接文本（把列表拼回一句话）
- 格式化输出（把变量塞进句子里）

## 2. 10 个高频操作（只讲“怎么用”）

## 2.1 看长度：`len`
```python
s = "abc123"
print(len(s))  # 6
```

## 2.2 取字符 / 取一段：索引与切片
```python
s = "abc123"
print(s[0])     # a
print(s[2:5])   # c12
print(s[::-1])  # 321cba
```

切片模板：`s[开始:结束:步长]`  
记忆：左闭右开（包含开始，不包含结束）。

## 2.3 查有没有：`find`（推荐先用它）
```python
s = "hello, world"
print(s.find("or"))  # 8
print(s.find("xx"))  # -1
```

找不到时：`find` 返回 `-1`，程序不会崩。

## 2.4 严格查找：`index`
```python
s = "hello, world"
print(s.index("or"))  # 8
# print(s.index("xx"))  # ValueError（会报错）
```

找不到时：`index` 会报错。  
你现在练习阶段优先用 `find` 更稳。

## 2.5 替换：`replace`
```python
s = "hello, world"
new_s = s.replace("world", "python")
print(s)      # hello, world
print(new_s)  # hello, python
```

重点：`replace` 不改 `s`，它返回新字符串。

## 2.6 去两端空白：`strip`
```python
s = "  hello  "
print(f"|{s.strip()}|")   # |hello|
print(f"|{s.lstrip()}|")  # |hello  |
print(f"|{s.rstrip()}|")  # |  hello|
```

口诀：
- `strip`：两边都去
- `lstrip`：只去左边（left）
- `rstrip`：只去右边（right）

补充：默认去空白字符（空格、`\n`、`\t`），也可以指定字符。

```python
s = "###hello###"
print(s.strip("#"))   # hello
print(s.lstrip("#"))  # hello###
print(s.rstrip("#"))  # ###hello
```

## 2.7 拆开：`split`
```python
s = "I#love#python"
parts = s.split("#")
print(parts)  # ['I', 'love', 'python']
```

`split` 是“字符串 -> 列表”。

## 2.8 拼回去：`join`
```python
parts = ['I', 'love', 'python']
text = "-".join(parts)
print(text)  # I-love-python
```

`join` 是“列表 -> 字符串”。  
谁调用，谁是连接符：`"-".join(parts)` 里 `-` 负责夹在中间。

## 2.9 大小写转换：`upper/lower/title`
```python
s = "hello world"
print(s.upper())  # HELLO WORLD
print(s.lower())  # hello world
print(s.title())  # Hello World
```

## 2.10 格式化输出：`f-string`（直接用它）
```python
name = "Spane"
score = 95.236
print(f"{name} 的成绩是 {score:.2f}")  # Spane 的成绩是 95.24
```

你现在只要先会这 3 个格式：
- `{变量}`：直接插值
- `{:.2f}`：保留两位小数
- `{:.2%}`：百分比

## 3. 你最近最容易踩的坑（重点）

1. 只写 `s.split("#")` 不接结果，以为 `s` 被改了。  
事实：`s` 不变，`split` 返回新列表。

2. 以为“不放 `print` 就没执行”。  
事实：执行了，只是你没看到结果（也没保存）。

3. 把 `find` 和 `index` 混用。  
`find` 找不到返回 `-1`；`index` 找不到直接报错。

4. 误以为能直接改字符串某个位置。  
`s[0] = "A"` 会报错，因为字符串不可变。

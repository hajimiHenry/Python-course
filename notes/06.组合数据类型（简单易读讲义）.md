## 2. 集合 `set`

### 2.1 集合是什么

集合可以理解成“**一堆不重复元素组成的无序容器**”。

它有两个最重要的特点：

- **无序**
  - 不能用下标取第几个元素
- **元素不重复**
  - 重复元素放进去后只保留一个

另外，集合里的元素必须是**不可变 / 可哈希**的值，比如：

- 可以放：`int`、`str`、`tuple`
- 不能放：`list`、`dict`、`set`

### 2.2 怎么创建集合

```python
a = {1, 2, 3}
b = set("pypy123")
c = {1, 2, 2, 3}
print(c)  # {1, 2, 3}
```

注意两个坑：

- 空集合不能写成 `{}`，因为 `{}` 是空字典
- 空集合必须写成 `set()`

```python
empty_set = set()
empty_dict = {}
```

### 2.3 常见操作

```python
A = {"p", "y", 123}
B = set("pypy123")

print(A - B)   # 差集
print(A & B)   # 交集
print(A | B)   # 并集
print(A ^ B)   # 对称差：只在其中一个集合里出现的元素
```

最常用的方法：

- `add(x)`：添加元素
- `discard(x)`：删除元素，元素不存在也不报错
- `remove(x)`：删除元素，元素不存在会报错
- `pop()`：随机弹出一个元素
- `clear()`：清空集合

### 2.4 集合最典型的用途

#### 用途 1：成员判断

```python
chars = {"p", "y", "t", "h", "o", "n"}
print("p" in chars)   # True
print("a" in chars)   # False
```

#### 用途 2：去重

```python
nums = [1, 2, 2, 3, 3, 3]
unique_nums = list(set(nums))
print(unique_nums)
```

注意：

- 去重后顺序通常会变
- 如果你既想去重又想尽量保留原顺序，不能只靠 `set`

### 2.5 这一部分你要记住什么

- 集合适合做“**去重**”和“**是否存在**”判断
- 集合没有顺序，不能按下标访问
- `{} ` 不是空集合，空集合要用 `set()`

## 3. 序列的共同特点

在学列表和元组之前，要先知道“序列”这个概念。

序列就是“**有先后顺序的一组元素**”，常见序列有：

- `str`
- `list`
- `tuple`

序列的共同能力：

- 可以用下标访问元素：`s[0]`
- 可以切片：`s[1:4]`
- 可以遍历：`for x in s`
- 可以求长度：`len(s)`
- 可以做成员判断：`x in s`

例如：

```python
ls = ["python", 123, ".io"]
print(ls[0])      # python
print(ls[-1])     # .io
print(ls[::-1])   # 反转
```

## 4. 列表 `list`

### 4.1 列表是什么

列表是最常用的序列类型，可以理解成“**可修改的有序容器**”。

特点：

- 有顺序
- 可重复
- 可修改
- 元素类型可以不同

### 4.2 怎么创建列表

```python
ls = ["cat", "dog", "tiger", 1024]
lt = list()
```

注意：

- 列表用 `[]` 或 `list()` 创建
- 不是 `set()`，这一点 PPT 里有一处写错了，别被带偏

### 4.3 列表和“引用”要分清

```python
ls = ["cat", "dog", "tiger", 1024]
lt = ls
```

这里不是复制出一个新列表，而是让 `lt` 和 `ls` 指向同一个列表对象。

所以改一个，另一个也会跟着变。

如果你想复制，常见写法有：

```python
lt = ls.copy()
# 或
lt = ls[:]
```

### 4.4 列表常见操作

```python
ls = ["cat", "dog", "tiger", 1024]

ls.append(1234)          # 末尾追加
ls.insert(1, "human")    # 指定位置插入
ls.remove("dog")         # 按值删除
del ls[0]                # 按下标删除
ls.reverse()             # 原地反转
ls.sort()                # 原地排序
```

其他高频操作：

- `len(ls)`：长度
- `max(ls)` / `min(ls)`：最大 / 最小值
- `ls.index(x)`：找元素下标
- `x in ls`：成员判断
- `ls.clear()`：清空列表

### 4.5 列表切片很灵活

```python
ls = ["cat", "dog", "tiger", 1024]
print(ls[1:3])     # 取下标 1 到 2
print(ls[::-1])    # 倒序
```

甚至还能用切片修改一段内容：

```python
ls = ["cat", "dog", "tiger", 1024]
ls[1:2] = [1, 2, 3, 4]
print(ls)
```

### 4.6 列表最典型的用途

#### 用途 1：动态管理一组数据

```python
courses = ["Python", "Java"]
courses.append("C")
courses.remove("Java")
```

#### 用途 2：遍历

```python
for item in courses:
    print(item)
```

#### 用途 3：排序

```python
nums = [12, 29, 9, 3]
print(sorted(nums))  # 返回新列表
```

区分一下：

- `ls.sort()`：直接改原列表
- `sorted(ls)`：返回排序后的新列表

### 4.7 这一部分你要记住什么

- 列表适合“**一批有顺序、会变化的数据**”
- 列表最常见的动作是增删改查
- 赋值不是复制，`lt = ls` 只是共用同一个列表

## 5. 实例：基本统计值计算

这部分的目标是：输入一组数字，计算它们的：

- 标准差

这样的好处是：**一个函数只做一件事**，比把所有逻辑塞到一起更清晰。

nums = []

### 5.4 这个例子真正要学的能力

## 6. 元组 `tuple`

### 6.1 元组是什么

元组也是序列，但它和列表最大的区别是：

- **元组一旦创建，就不能修改**

可以理解成“**只读列表**”。

### 6.2 怎么创建元组

```python
t1 = ("cat", "dog", "tiger")
t2 = tuple([1, 2, 3])
t3 = 1, 2, 3
```

Python 里很多时候即使不写括号，也会自动把多个值打包成元组。

```python
def func():
    return 1, 2
```

### 6.3 元组能做什么

元组继承了序列的大部分通用操作：

```python
creature = "cat", "dog", "tiger", "human"
print(creature[::-1])
print(creature[1])
```

但它不能像列表那样：

- `append`
- `remove`
- 修改某个位置的值

### 6.4 元组最典型的用途

#### 用途 1：函数返回多个值

```python
def calc(x, y):
    return x + y, x - y, x * y

result = calc(9, 2)
print(result)  # (11, 7, 18)
```

#### 用途 2：表示“不希望被改动”的一组数据

比如：

- 坐标
- RGB 颜色
- 数据库查询结果中的固定记录

### 6.5 这一部分你要记住什么

- 元组是“**有序但不可修改**”的序列
- 适合表达固定数据
- 函数返回多个值时，本质上常常就是返回一个元组

## 7. 字典 `dict`

### 7.1 字典是什么

字典是 Python 最重要的数据结构之一，可以理解成“**键和值的对应关系**”。

比如：

```python
{
    "中国": "北京",
    "美国": "华盛顿"
}
```

这里：

- `"中国"`、`"美国"` 是键（key）
- `"北京"`、`"华盛顿"` 是值（value）

### 7.2 字典和列表的区别

列表是靠“位置”取值：

```python
ls[0]
```

字典是靠“键”取值：

```python
d["中国"]
```

所以字典特别适合表示：

- 姓名 -> 成绩
- 国家 -> 首都
- 单词 -> 词频

### 7.3 怎么创建字典

```python
d = {"中国": "北京", "美国": "华盛顿", "法国": "巴黎"}
empty = {}
```

也可以用 `dict()` 创建。

### 7.4 字典常见操作

```python
d = {"中国": "北京", "美国": "华盛顿"}

print(d["中国"])            # 按键取值
d["法国"] = "巴黎"          # 新增
d["美国"] = "Washington"    # 修改
print("中国" in d)          # 判断键是否存在
print(len(d))               # 键值对个数
print(d.keys())             # 所有键
print(d.values())           # 所有值
print(d.items())            # 所有键值对
```

很重要的方法：

```python
print(d.get("中国", "默认值"))
print(d.get("日本", "东京"))
```

`get()` 的好处是：

- 键存在时，返回对应值
- 键不存在时，不报错，而是返回默认值

这在做统计时特别好用。

### 7.5 字典最典型的用途

#### 用途 1：表达映射关系

```python
student = {
    "name": "Alice",
    "age": 18,
    "major": "CS"
}
```

#### 用途 2：做计数统计

```python
ls = [1, 2, 1, 3, 4, 2, 1]
counts = {}
for x in ls:
    counts[x] = counts.get(x, 0) + 1
print(counts)
```

这句一定要会：

```python
counts[x] = counts.get(x, 0) + 1
```

意思是：

- 如果 `x` 之前没出现过，就从 `0` 开始记
- 如果出现过，就在原来的基础上加 `1`

### 7.6 这一部分你要记住什么

- 字典是“**键 -> 值**”的映射
- 字典最适合做查找和统计
- 做计数时，`get()` 非常关键

## 8. 实例：文本词频统计

这是这一章最重要的综合应用之一。

目标是：给一段文本，统计每个词出现了多少次。

### 8.1 英文文本词频统计的基本流程

1. 读入文本
2. 把大小写统一成小写
3. 去掉标点符号
4. 按空格切分成单词列表
5. 用字典统计每个单词出现次数
6. 排序，输出前几名

核心代码思路：

```python
counts = {}
for word in words:
    counts[word] = counts.get(word, 0) + 1
```

排序思路：

```python
items = list(counts.items())
items.sort(key=lambda x: x[1], reverse=True)
```

这里 `x[1]` 表示按“出现次数”排序。

### 8.2 中文文本为什么更麻烦

英文天然有空格：

```text
I love Python
```

中文通常没有天然空格：

```text
我喜欢Python
```

所以中文文本统计词频前，得先做一件事：

- **分词**

也就是先把一句连续的中文切成一个个词。

## 9. `jieba` 和中文分词

### 9.1 `jieba` 是什么

`jieba` 是一个常用的中文分词第三方库。

安装命令：

```bash
pip install jieba
```

### 9.2 最常用的函数

最常用的是：

```python
jieba.lcut(text)
```

它会直接返回一个列表。

例如：

```python
import jieba

text = "我喜欢学习Python"
words = jieba.lcut(text)
print(words)
```

### 9.3 中文词频统计的思路

```python
import jieba

txt = open("threekingdoms.txt", "r", encoding="utf-8").read()
words = jieba.lcut(txt)
counts = {}

for word in words:
    if len(word) == 1:
        continue
    counts[word] = counts.get(word, 0) + 1
```

这里过滤掉单个字，是因为很多单字没有分析价值。

### 9.4 面向问题继续优化

如果目标不是“统计所有词”，而是“统计《三国演义》人物出场次数”，那就还要继续处理：

- 去掉无意义高频词，比如“将军”“却说”
- 把同一个人的不同叫法合并

例如：

- `"诸葛亮"` 和 `"孔明曰"` 归成 `"孔明"`
- `"关公"` 和 `"云长"` 归成 `"关羽"`

这说明一个很重要的道理：

- **程序不是写完就结束，结果出来后还要继续根据问题优化**

## 10. 四种数据类型怎么选

如果你做题时总分不清该用哪个，先看下面这张“人话版”判断表：

- 需要去重、做交并差、判断某元素在不在：用 `set`
- 需要按顺序保存一批数据，而且后面还会改：用 `list`
- 需要按顺序保存一批固定数据，不想让它变：用 `tuple`
- 需要“名字 -> 内容”这样的对应关系，或者要做计数：用 `dict`

你可以这样记：

- `set`：查重保安
- `list`：可改清单
- `tuple`：封口清单
- `dict`：带索引的登记表

## 12. 复习时最容易错的点

- 空集合是 `set()`，不是 `{}`
- 列表用 `[]` 或 `list()` 创建，不是 `set()`
- 集合不能放 `list`、`dict` 这种可变对象
- `lt = ls` 不是复制列表，只是共用同一个对象
- 元组不能修改
- 字典判断的是“键是否存在”，不是“值是否存在”
- `d.get(key, 0)` 比直接写 `d[key]` 更适合做统计
- 中文词频统计前通常先分词
